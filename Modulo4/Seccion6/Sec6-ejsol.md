## **Ejercicio 1**  

Cuál es el resultado del siguiente fragmento de código?  
```python
from datetime import time

t = time(14, 39)
print(t.strftime("%H:%M:%S"))
```

```
14:53:00
```

<br></br>  

## **Ejercicio 2**  

Cuál es el resultado del siguiente fragmento de código?  
```python
from datetime import datetime

dt1 = datetime(2020, 9, 29, 14, 41, 0)
dt2 = datetime(2020, 9, 28, 14, 41, 0)

print(dt1 - dt2)
```

```
1 day, 0:00:00
```
