## **Ejercicio 1**  

Cuál es el resultado del siguiente fragmento de código?  
```python
import calendar
print(calendar.weekheader(1))
```


## **Ejercicio 2**  

Cuál es el resultado del siguiente fragmento de código?  
```python
import calendar  

c = calendar.Calendar()

for weekday in c.iterweekdays():
    print(weekday, end=" ")
```

```
0 1 2 3 4 5 6
```
